# NextOpsAgent Screenshot Guide

Screenshots are pending and should be captured manually before final README image
links are enabled.

Use the bundled fictional sample data only. Recommended browser width: about
1440 px desktop. Keep the Streamlit sidebar visible for at least the home hero
shot, because it shows the local-first safety boundary.

Recommended sample workflow for all screenshots:

`WF-001 - Tutoring business`

This sample clearly shows WeChat intake, Excel tracking, manual follow-up,
response delays, and weekly reporting.

## Safety review before public use

- Confirm all visible workflows are fictional.
- Do not show real names, emails, phone numbers, private chats, API keys, or credentials.
- Hide unrelated browser tabs, notifications, and personal desktop information.
- Review every screenshot at full size before adding it to a public portfolio page.
